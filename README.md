# 📱 Telegram ID Lookup Tool

A demo educational project interface for Telegram account lookup systems built using **Python** and **Telegram API** concepts.

> ⚠️ This repository is created only for educational and UI demonstration purposes.

---

# 🚀 Features

✅ Telegram Username Search  
✅ Demo Profile Lookup  
✅ Fast Python Backend  
✅ Beginner Friendly  
✅ Secure Environment Variables  
✅ Open Source Project  
✅ Heroku & Render Ready

# ⚡ Requirements

- Python 3.10+
- Telegram API
- Pyrogram

---

# ⚙️ Environment Variables

Create `.env`

```env
API_ID=12345678
API_HASH=your_api_hash
BOT_TOKEN=your_bot_token
```

---

# 📜 requirements.txt

```txt
pyrogram
tgcrypto
python-dotenv
```

---

# ▶️ Run Project

```bash
python main.py
```

---

# ☁️ Deploy on Heroku

```bash
heroku login
heroku create telegram-id-lookup
git push heroku main
```
